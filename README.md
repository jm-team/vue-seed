# vue-pc-seed

> vue2 pc版项目脚手架

## 简单几步构建基础项目

```
# 安装依赖
npm install

# 启动开发服务 http://localhost:8080
npm run dev

# 构建项目
npm run build
```

## 简介
* 加入单点登录功能
* 加入公共底部在内容加载后显示优化
* 加入图片懒加载
* 引入element UI组件库
* 抽离了公共api接口和公共js方法
* 使用json模拟的数据，部分功能不能用，可以查看测试环境：http://www.jtt.com
* 登录是模拟的，随便输入就能登录，无法保持登录状态
